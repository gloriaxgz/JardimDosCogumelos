Sobre Este Site

Este site foi criado de maneira amadora, como um simples projeto universitário. Grande parte dos textos foram gerados através do ChatGPT e, por isso, é importante notar que os textos, apesar de trazerem informações verdadeiras, não foram produzidos com métodos muito refinados ou precisos, uma vez que o foco aqui é o estudo de desenvolvimento web.
Para o design, foi utilizado um template que pode ser encontrado na internet, no site html Design, linkado no rodapé desta página.
O template utilizado se chama "NLW – Lawncare free garden website template" e foi alterado com imagens de bancos de dados e outras obtidas diretamente do Google Imagens que, portanto, podem ter direitos autorais.
Contudo, este projeto não é, de nenhuma forma, lucrativo.
Por último, mas não menos importante, vale dizer que o tema deste site, os cogumelos, foi inspirado não por gosto pessoal meu, mas sim de pessoas, do meu convívio, que admiro.
